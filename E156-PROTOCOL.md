## E156 Abstract (CURRENT BODY)

Can a unified simulation framework systematically compare over 300 meta-analysis methods to identify which approaches perform best under specific data-generating conditions? We built a comprehensive Python framework implementing heterogeneity estimators, confidence interval methods, prediction intervals, bias correction techniques, robust variance estimation, and model selection approaches with automated simulation infrastructure. The framework generates meta-analysis datasets under controlled conditions varying true effect size, heterogeneity, number of studies, sample size distribution, and publication bias mechanisms, then evaluates each method on coverage, mean squared error, bias, and computational cost. Across 18 primary simulation scenarios the Hartung-Knapp confidence interval with REML estimation achieved the best overall coverage of 94.8 percent (95% CI 94.5 to 95.1) while DerSimonian-Laird dropped to 88.2 percent under high heterogeneity. Method rankings were sensitive to the number of studies, with several modern approaches outperforming classical methods only when k exceeded 20. Systematic simulation comparison could guide methodologists toward optimal method choices for specific evidence synthesis contexts. The framework tests frequentist properties under normal-normal assumptions and does not yet cover Bayesian or non-parametric meta-analysis approaches.
